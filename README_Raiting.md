<h1>⭐ Рейтинг и сборы — плагин для Lampa</h1>
<p>Плагин улучшает отображение рейтингов фильмов и сериалов: на карточках (главный экран, списки) и на странице фильма. Поддерживает множество источников: TMDb, IMDb, КиноПоиск, Rotten Tomatoes, Metacritic, Trakt, Letterboxd, Roger Ebert, MyAnimeList. Также показывает кассовые сборы (бюджет, мировые, США, Россия) и количество наград (Оскары).</p>

<h2>✨ Возможности</h2>
<ul>
    <li>Выбор основного рейтинга для карточек (TMDB, IMDb, КиноПоиск, Rotten Tomatoes, Metacritic, Trakt)</li>
    <li>На странице фильма отображаются все доступные рейтинги (основные и дополнительные)</li>
    <li>Кэширование данных (настраиваемый TTL) для экономии запросов к API</li>
    <li>Поддержка нескольких API: MDBList, OMDB, КиноПоиск Unofficial</li>
    <li>Отображение бюджета и сборов (если доступны)</li>
    <li>Количество побед на премиях (Оскары) из OMDB</li>
    <li>Полностью настраивается: можно включить/выключить отдельные рейтинги и сборы</li>
    <li>Автоматическое форматирование значений (100-балльные -> целые, 10-балльные -> одна десятая)</li>
</ul>

<h2>📥 Установка</h2>
<ol>
    <li>Скачайте файл плагина (например, <code>rait.js</code>) и разместите его на своём сервере или в репозитории GitHub.</li>
    <li>В приложении Lampa откройте <strong>Настройки → Расширения → Добавить плагин</strong>.</li>
    <li>Вставьте прямую ссылку на файл плагина.</li>
    <li>После установки в настройках Lampa появится пункт «Рейтинг».</li>
</ol>

<div class="info">
    <strong>Важно:</strong> Для получения дополнительных рейтингов (Rotten Tomatoes, Metacritic, Trakt и др.) требуется API-ключ MDBList. Для кассовых сборов КиноПоиска — ключ КиноПоиска Unofficial. Для резервного получения RT/Meta — ключ OMDB. Все ключи вводятся в настройках плагина. Без них будут отображаться только базовые рейтинги (TMDB/IMDb/КП), если они уже есть в карточке.
</div>

<h2>⚙️ Настройки</h2>
<p>Плагин добавляет раздел «Рейтинг» в настройки Lampa со следующими параметрами:</p>
<ul>
    <li><strong>Рейтинг в карточках (главная, списки)</strong> — выбор источника для отображения на карточках (TMDB, IMDb, КиноПоиск, Rotten Tomatoes, Metacritic, Trakt).</li>
    <li><strong>Дополнительные рейтинги на странице фильма</strong> — можно включить/отключить показ Letterboxd, Roger Ebert, MyAnimeList, количество Оскаров.</li>
    <li><strong>API ключ MDBList (Все рейтинги и Trakt)</strong> — основной ключ для получения большинства рейтингов.</li>
    <li><strong>API ключ Кинопоиска (Детальные сборы)</strong> — для получения бюджета и сборов с КиноПоиска.</li>
    <li><strong>API ключ OMDB (Резервный для RT/Meta)</strong> — используется, если MDBList не вернул RT/Meta.</li>
    <li><strong>TTL кэша (дни)</strong> — время хранения кэша (7, 14 или 30 дней).</li>
    <li><strong>🧹 Очистить кэш рейтингов и сборов</strong> — мгновенно удаляет все сохранённые данные.</li>
</ul>

<h2>📊 Как это работает</h2>
<ol>
    <li>При просмотре карточек на главном экране или в списках плагин проверяет, есть ли уже рейтинг в данных карточки. Если нужного источника нет — запрашивает его через MDBList или OMDB (при наличии ключей). Результат кэшируется.</li>
    <li>На странице фильма плагин собирает все доступные рейтинги и отображает их в виде цветных плашек (TMDB, IMDb, КП, RT, Meta, Trakt и опциональные).</li>
    <li>Параллельно запрашиваются кассовые сборы (бюджет, мировые, США, Россия) через КиноПоиск Unofficial или MDBList. Они отображаются строкой под рейтингами.</li>
    <li>Если OMDB возвращает информацию о наградах, выводится количество побед (Оскары).</li>
</ol>

<h2>🖥️ Пример отображения</h2>
<p>На странице фильма строка рейтингов может выглядеть так:</p>
<pre><code> 7.8 TMDB    7.5 КП    7.2 IMDb    85% RT    74 Meta    67 Trakt</code></pre>
<p>Ниже может быть строка сборов:</p>
<pre><code>Бюджет: $150.0 млн   Мир: $500.0 млн   США: $200.0 млн</code></pre>
