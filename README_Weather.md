<h1>🌤️ Weather Widget — виджет погоды для Lampa</h1>
<p><strong>Плагин добавляет виджет погоды в интерфейс Lampa.</strong> Показывает текущую температуру, почасовой прогноз, детальную информацию об условиях, а также красивую модальную панель с расширенными данными. Использует бесплатное API <a href="https://www.weatherapi.com" style="color:#ffb347;">WeatherAPI.com</a>.</p>

<h2>✨ Возможности</h2>
<ul>
    <li>Отображение текущей температуры и иконки погоды в шапке Lampa</li>
    <li>Модальное окно с подробностями:
        <ul>
            <li>Почасовой прогноз (до 10 часов вперёд) с анимированными фонами (день/ночь/рассвет/закат)</li>
            <li>Текущие условия: температура, ощущается как, ветер, влажность, давление, облачность, дождь, УФ-индекс и т.д.</li>
            <li>Прогноз на день: мин/макс температура, фаза луны, восход/закат, освещённость луны</li>
        </ul>
    </li>
    <li>Настройки: можно указать свой API-ключ и город (по умолчанию Санкт-Петербург)</li>
    <li>Кэширование данных на 6 часов (экономит запросы к API)</li>
    <li>Автоматическое обновление при смене часа</li>
    <li>Работает на телевизорах и мобильных устройствах (адаптивная верстка)</li>
</ul>

<h2>📥 Установка</h2>
<ol>
    <li>Скачайте файл плагина (например, <code>weather.js</code>) и разместите его на своём сервере или в репозитории GitHub.</li>
    <li>В приложении Lampa откройте <strong>Настройки → Расширения → Добавить плагин</strong>.</li>
    <li>Вставьте прямую ссылку на файл плагина.</li>
    <li>После установки в настройках Lampa появится пункт «Виджет погоды».</li>
</ol>

<div class="info">
    <strong>Важно:</strong> Для работы необходим бесплатный API-ключ от <a href="https://www.weatherapi.com/signup.aspx" style="color:#ffb347;">WeatherAPI.com</a>. Получить его можно за 1 минуту. В настройках плагина есть кнопка «Получить API» для перехода на сайт регистрации.
</div>

<h2>⚙️ Настройки</h2>
<p>После установки доступны следующие параметры в разделе «Виджет погоды»:</p>
<ul>
    <li><strong>API ключ</strong> — ваш ключ от WeatherAPI. Без него данные не загрузятся.</li>
    <li><strong>Город</strong> — название населённого пункта (можно писать по-русски или по-английски).</li>
    <li><strong>Получить API</strong> — кнопка для перехода на сайт WeatherAPI.</li>
</ul>

<h2>📊 Как это работает</h2>
<ol>
    <li>Плагин отправляет запрос к <code>api.weatherapi.com/v1/forecast.json</code> с вашим ключом и названием города.</li>
    <li>Ответ кэшируется в локальном хранилище Lampa на 6 часов (чтобы не превысить лимит бесплатного тарифа — 1 млн запросов в месяц).</li>
    <li>Каждые полчаса данные обновляются из кэша, а раз в 6 часов — свежий запрос к API.</li>
    <li>При смене часа автоматически подгружаются новые данные.</li>
    <li>Виджет в шапке показывает текущую температуру и иконку. При нажатии открывается модальное окно с полной информацией.</li>
</ol>

<h2>🖥️ Пример интерфейса</h2>
<p>После установки в шапке Lampa появится виджет:</p>
<pre><code>[Температура: 22°] [Иконка погоды]</code></pre>
<p>При клике открывается окно с тремя вкладками: «Текущая», «Следующий час», «День». Навигация между вкладками — стрелками влево/вправо. Выход — кнопка «Назад».</p>

<h2>🤝 Благодарности</h2>
