<h1>▶️ Продолжить просмотр (Continue Watching)</h1>
<p><strong>Плагин для Lampa</strong>, который добавляет кнопку «Продолжить» в карточку фильма или сериала. Использует данные lampac (истории просмотров) для мгновенного перехода к последней просмотренной серии.</p>

<h2>✨ Возможности</h2>
<ul>
    <li>Автоматически определяет последний просмотренный эпизод (даже если смотрели через внешний плеер)</li>
    <li>Показывает процент прогресса (например, «Продолжить S2E3 (72%)»)</li>
    <li>Настраиваемый текст кнопки: можно скрыть/показать сезон, балансер, озвучку</li>
    <li>Гибкое позиционирование: кнопку можно поставить первой, после «Смотреть» или в любое другое место</li>
    <li>Работает только при наличии сохранённой истории lampac (ничего не сломает, если данных нет)</li>
    <li>Не требует дополнительных настроек lampac — использует уже существующие ключи </li>
</ul>

<h2>📥 Установка</h2>
<ol>
    <li>Скопируйте содержимое файла <code>btn.js</code> (или весь код плагина).</li>
    <li>В приложении Lampa откройте <strong>Настройки → Расширения → Добавить плагин</strong>.</li>
    <li>Вставьте прямую ссылку на файл <code>btn.js</code> (или загрузите его на свой сервер/хостинг).</li>
    <li>После установки плагин появится в разделе <strong>«Продолжить просмотр»</strong> в настройках Lampa.</li>
</ol>

<h2>⚙️ Настройки</h2>
<p>После установки доступны следующие параметры:</p>
<ul>
    <li><strong>Позиция кнопки</strong> — порядковый номер, куда вставить кнопку. Тут всё сложно... должно быть так но не факт))))) 1 = последняя, 2 = первая, 3 = вторая(или нет) и т.д.</li>
    <li><strong>Показывать сезон/серию</strong> — добавлять «S01E03» к тексту кнопки.</li>
    <li><strong>Показывать балансер</strong> — добавлять название источника (например, Filmix).</li>
    <li><strong>Показывать озвучку</strong> — добавлять название озвучки (например, Дубляж).</li>
</ul>

<p>Логика плагина: <strong>берём эпизод с наибольшим номером, у которого есть ненулевое время или процент просмотра</strong>. Поэтому откроется <strong>Эпизод 4</strong>, а не 3 или 2. Даже если Эпизод 3 начат позже, но по нумерации он меньше, приоритет отдаётся самому новому просмотренному номеру.</p>
<p>Все настройки применяются сразу, без перезагрузки Lampa.</p>

<h2>🧠 Как это работает</h2>
<ol>
    <li>При открытии карточки фильма/сериала плагин проверяет <code>online_watched_last</code> для получения последнего балансера и последней серии.</li>
    <li>Если данные есть — рисуется кнопка «Продолжить». При нажатии открывается lampac с нужным источником.</li>
    <li>После загрузки списка эпизодов плагин ищет среди них тот, который имеет <strong>наибольший номер</strong> и ненулевой прогресс в <code>file_view</code>. Именно он считается последним просмотренным.</li>
    <li>На этот эпизод автоматически отправляется <code>hover:enter</code> — запускается плеер с сохранённой позицией.</li>
</ol>
